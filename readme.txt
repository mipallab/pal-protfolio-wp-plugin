== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pla Protfolio -> Add item.
4. create a page or where you use than past this code '[pla-protfolio]'
5. save or update the page.
6. reload the page
7. enjoy our plugin.
